Machine Learning High

A subfield of artificial intelligence that enables systems to learn and improve from experience without being explicitly programmed. It focuses on developing algorithms that can access data and use it to learn for themselves.

Machine Learning Pipeline Medium

A series of data processing steps that automate the workflow of a machine learning model, from raw data ingestion through feature engineering, training, evaluation, and deployment.

Supervised Machine Learning Low

A type of machine learning where the model is trained on labeled data — each training example is paired with the correct output — so the algorithm learns to map inputs to outputs.
